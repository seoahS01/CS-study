## 진행 방식 및 정리 가이드
> 각 팀원은 배정된 질문에 대해 탐구하고 질문에 대한 **깔끔한 정답**을 도출하는 것이 목표입니다.

- 발표 시간: 팀원당 **10분** 내외
- 자료 기록: 각 카테고리 폴더 내에 `(질문).md` 파일 형식을 정리해 업로드 합니다. (PDF 도 좋습니다.)
- **필수 포함 요소** (최소 1개 이상)

    - 이해를 돕는 시각적 자료(아키텍처, 순서도, 사진 등)
    - 구체적 예시 코드
    - 개념을 쉽게 설명할 수 있는 비유나 예제
    - 또는 이에 준하는 것
  
- 마지막에는 필수적을 질문에 대한 **1분 이내의 답변**을 **깔끔**하게 준비해야 합니다.

## 🤔질의 응답 시간 및 피드백

- 모든 조원은 해당 주제와 관련된 질문을 **각 1개씩** 만듭니다.(필수)
- 발표자는 이에 대한 답변을 합니다.
